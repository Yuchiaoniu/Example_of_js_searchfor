# _searchfor功能介紹_

在home/index.html中登入google會員後，輸入客製化title<br>
在edit/<a>yourtitle</a>.html中修改文字和圖片，儲存頁面後跳轉到paypal付款頁面<br>
在page/<a>yourtitle</a>.html顯示遞交產品<br>

## 使用API<br>
firebase:auth
paypal:subscribion API、product API、IPN API

## 使用技術<br>
html, css, js, php
上傳文字:AJAX、php
上傳圖片:imgcropper、php
結帳:node.js
